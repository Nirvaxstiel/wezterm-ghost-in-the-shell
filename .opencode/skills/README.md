# Skills

Specialized skills that the agent can use for specific tasks.

## Available Skills

| Skill                       | Description                                                        |
| --------------------------- | ------------------------------------------------------------------ |
| **code-agent**              | Debug, implement, edit code with context awareness                 |
| **analysis-agent**          | Code review, evaluation, and quality assessment                    |
| **research-agent**          | Investigation and information gathering                            |
| **git-commit**              | Conventional commit message generation                             |
| **pr**                      | Pull request description and title writing                         |
| **rlm**                     | Recursive context management for large tasks                       |
| **self-learning**           | Self-improvement and pattern recognition                           |
| **code-review**             | Structured code review workflow with priority-based classification |
| **prompt-engineer**         | Prompt engineering best practices and safety frameworks            |
| **security-audit**          | OWASP-based security audit workflow                                |
| **intent-classifier**       | Query classification and routing logic                             |
| **skill-composer**          | Dynamic skill composition for complex tasks                        |
| **workflow-management**     | Production-grade development workflow                              |
| **formatter**               | Code quality cleanup and formatting                                |
| **context7**                | Live documentation retrieval via Context7 API                      |
| **context-manager**         | Context discovery, extraction, and management                      |
| **task-tracking**           | Progressive tracking system for tasks                              |
| **verifier-code-agent**     | Generator-Verifier-Reviser pattern for reliable code               |
| **reflection-orchestrator** | Explicit self-verification and adversarial critique                |
| **model-aware-editor**      | Model-specific edit format optimization                            |

## Usage

Skills are automatically loaded based on the detected intent and task requirements.

See the parent [README](../../README.md) for more information on how the system works.
